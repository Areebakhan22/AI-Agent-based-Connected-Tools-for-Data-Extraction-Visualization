# SysIDE Modeler - Free Trial Setup

## 30-Day Free Trial Available

SysIDE Modeler offers a **30-day free trial** with full features including diagram visualization.

## How to Activate Trial Mode

### Method 1: Automatic Prompt
1. Open `OpsCon.sysml` in Cursor
2. Press `Ctrl+Shift+V` (or `Ctrl+Shift+P` → "Syside Modeler: Visualize current file")
3. The extension will prompt you to set up a license
4. **Select "Trial" or "Evaluation" mode** when prompted
5. Follow the setup wizard

### Method 2: Manual License Check
1. Open `OpsCon.sysml` in Cursor
2. Press `Ctrl+Shift+P` (Command Palette)
3. Run: **"SysIDE Modeler: Check Syside license"**
4. If no license found, it will guide you to trial setup

### Method 3: Import Trial License
1. Press `Ctrl+Shift+P`
2. Run: **"SysIDE Modeler: Import Syside license"**
3. Choose "Trial" or "Evaluation" option
4. Follow the prompts

## Get Trial License Key (if needed)

If you need a trial license key:
1. Visit: https://sensmetry.com/syside-pricing/
2. Select "Solo" plan
3. Click "Free Trial"
4. Register and get your trial license key
5. Use "Import Syside license" command to enter it

## Verify Trial is Active

1. Press `Ctrl+Shift+P`
2. Run: **"SysIDE Modeler: Check Syside license"**
3. Should show trial status and expiration date

## Troubleshooting

- **License prompt not appearing**: Make sure `OpsCon.sysml` is open and active
- **Extension not activating**: Reload window (`Ctrl+Shift+P` → "Reload Window")
- **Commands not visible**: Ensure extension is enabled in Extensions view

## Note

The trial provides full functionality for 30 days. After that, you can:
- Purchase a license
- Continue with limited features (if available)
- Use your existing Python slides generator (works independently)
